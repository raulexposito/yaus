## 2-Minutes Guide:

Install Maven and Oracle JDK if you haven't done it yet.

```
$ git clone https://github.com/raulexposito/yaus.git
Cloning into 'yaus'...
...
$ cd yaus/
$ mvn package
$ cd web/
$ mvn jetty:run
```

Open your favorite browser on [http://localhost:8080/](http://localhost:8080/)
